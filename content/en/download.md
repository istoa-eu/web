With the *iStoa* simulation environment, you can explore DKMs with the
script examples already provided or edit new ones to invent
variations.

*iStoa* is also a development environment, so you can study and modify
the DKM code as you wish.

* [GNU/Linux](https://github.com/istoa-eu/app/releases/latest)
* [Windows](https://github.com/istoa-eu/app/releases/latest)
* [Mac OS X](https://github.com/istoa-eu/app/releases/latest)
